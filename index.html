How does the Internet work?

While many may be familiar with what the the internet can do, not many know how it works. This site will give you a better idea of the different parts of the internet and how they all work together to give us the internet that we know all to well.

Domain Names and ICANN

Domain names are part of a website URL that identifies a particular site. ICANN is known as the Internet Corporation for Assigned Names and Numbers and is responsible for coordinating technical maintenance of name and numerical spaces; such as domain names and IP addresses on the internet.

Learn more
IP Addresses, Packets, and Routing

IP addresses are the unique numerical identifications assigned to every device connected to the internet. Packets contain the information being routed from one device or network to another as well as the sender and receiving IP addresses. IP routing is the route that packets will take to arrive at their final destination.

Learn more
DNS

DNS is the Domain Naming System which basically acts as a phone book for domain names and IP addresses on the the internet.

Learn more
HTTP and HTTPS Protocols

HTTP is the Hypertext Transfer Protocol and it sets the rules that dictate how information is transmitted on the web. HTTPS is the Hypertext Transfer Protocol Secure which is a more secure and advanced version of HTTP.

Learn more
W3C, HTML, and CSS

World Wide Web Consortium (W3C) is an international organization that is committed to ensuring the internet is continuously improving and growing HTML and CSS are technologies to build web pages.
