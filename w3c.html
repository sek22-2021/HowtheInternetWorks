W3C, HTML, and CSS

World Wide Web Consortium (W3C)

W3C or the World Wide Web Consortium is an international community devoted to developing Web standards and to lead the Web to its full potential. It is led by Tim Berners-Lee, inventor of the World Wide Web, and CEO, Jeffrey Jaffe.

Tim Berners-Lee is a computer scientist who is best known for inventing the world wide web. He invented the W3C with the main goal of improving the quality of the web. He kept the idea open to be freely available and adoptable by anyone. Transparency is one of their fundamental principles of operation. The W3C developed standards such as HTML, XHTML, CSS, HTTP, CGI and more.

HTML

HTML is known as Hyper Text Markup Language and essentially tells a web browser how to display certain pages. It is the language of interfaces for websites and it used to customize text and either request image URLs or post requesting certain information like login information. HTML code uses markdown software such as Pycharm and Webstorm along with associated tags to illustrate the task it would like to carry out.

CSS

CSS, or cascading style sheet, is the language used to describe the presentation of a website such as its fonts, font sizes, colors, images and overall layout. It adapts the presentation based on the wide variety of devices which may access it. CSS is similar to HTML in that it also customizes the look of a webpage however HTML deals more with the text while CSS focuses more on visual effects and overall layout.
